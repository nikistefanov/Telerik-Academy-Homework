XML - XML stands for EXtensible Markup Language. It is widely used for the representation of arbitrary data structures[6] such as those used in web services. XML was designed to store and transport data and was designed to be both human- and machine-readable.

XML Namespace - An XML namespace is a collection of names, identified by a Uniform Resource Identifier (URI) reference, which are used in XML documents as element and attribute names. XML namespace provide a method to avoid element name conflicts.
